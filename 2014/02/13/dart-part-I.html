Tour de Dart (Part I)

13 Feb 2014 by Nane Kratzke

This (german) presentation provides a comprehensive survey of the programming language Dart and its core concepts. Dart is a web programming language mainly developed by Google with the intention to provide an open and modern alternative to JavaScript (the actual de facto standard in web client side programming). To understand the survey you should be familiar with at least one programming language and the concepts of object oriented programming.

This Part I concentrates on the language Dart itself and its

  • optional type system,
  • data types,
  • functions,
  • operators,
  • object oriented concepts,
  • and generics.

Upcoming part II will concentrate on further aspects of dart.

  • library system
  • asynchronous programming
  • I/O programming
  • manipulating the DOM tree
  • server/client side web programming
  • conversion of data formats (often used in the web domain)

So stay tuned.
