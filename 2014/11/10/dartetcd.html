dartetcd - A Dart API for ETCD

10 Nov 2014

Tags

  • #blog
  • #dart
  • #programming

Share

etcd is a highly-available key value store for shared configuration and service discovery. It has been developed by CoreOS to manage Docker container orchestration. etcd is inspired by Apache ZooKeeper, with a focus on being simple (curl'able API using HTTP+JSON), secure, fast and reliable.

etcd is written in Go and uses the Raft consensus algorithm to manage a highly-available replicated log.

dartetcd provides a Dart library to interface with etcd. From a very high level point of view etcd provides a distributed map (properly distributed using Raft consensus algorithm). You will learn

  • how to use dartetcd in Dart projects
  • how to list keys
  • how to set key, value pairs (files)
  • how to modify key, value pairs
  • how to create and delete directories
  • how to observe directories and files for change.

Use dartetcd in Dart projects

You simply have to import the dartetcd library and you are ready to use etcd from Dart. The following snippet will create a etcd object, connect to an etcd node and will retrieve all registered keys from the distributed key value store.

import 'package:etcd/etcd.dart';

void main() {

  ETCD etcd = new ETCD.connect("192.168.59.103", 4001);
  etcd.list().then((List<String> keys) {
   keys.forEach((key) => print(key));
  });

}

How to list keys

/**
 * Method to get post parameters from a [Request] object.
 */
Future<Map<String, String>> getPostParams({ Encoding enc: UTF8 }) {
  Completer c = new Completer();
  this.input.transform(const AsciiDecoder()).listen((content) {
    final postParams = new Map.fromIterable(
        content.split("&").map((kvs) => kvs.split("=")),
        key: (kv) => Uri.decodeQueryComponent(kv[0], encoding: enc),
        value: (kv) => Uri.decodeQueryComponent(kv[1], encoding: enc)
    );
    c.complete(postParams);
  });
  return c.future;
}
