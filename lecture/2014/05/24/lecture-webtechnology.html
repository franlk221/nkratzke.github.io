The lecture Webtechnology is about common technologies used to build (part) of web information systems. Lecture will be held for computer science students (Computer Science/Software Engineering study programme, B. Sc.) at LÃ¼beck University of Applied Sciences. Lecture will cover following aspects:

  • IaaS Cloud Computing to host (distributed) web applications
  • descriptional languages like HTML
  • presentational languages like CSS
  • Document Object Model (DOM) used for example in web browsers
  • client side programming with languages like Dart or JavaScript
  • server side programming with languages like Dart or PHP
  • security aspects like HTML/SQL injections or session hijacking

Practical Course

Students learn how to apply these techniques in an accompanying practical course. This course will cover descriptional languages like HTML and CSS and will have a clear focus on client and server side programming using Dart as well as on hosting applications with IaaS cloud computing means (see sponsors below).

In small groups students

  • develop a variant of the famous Snake game concept,
  • derive a MVC based architecture to realize their game concept and
  • realize their game by using HTML/CSS as well as Dart to implement necessary model, view and controller components.

Course Material (german only)

  • Crash course: IaaS Cloud Computing
  • Lecture Unit 1: HTML and CSS
  • Lecture Unit 2: Dart I (aka Tour de Dart Part I)
  • Lecture Unit 3: Dart II (aka Tour de Dart Part II)
  • Exercise 1: HTML and CSS
  • Exercise 2: Programming Dart
  • Exercise 3: Client/Server side programming with Dart

Example Applications (Github)

  • Reference implementation of the (websocket) server based Dartchat can be found on Github
