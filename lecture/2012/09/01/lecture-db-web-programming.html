Database-based Web Server Programming (Winter 2012)

The lecture is about common technologies used to build (part) of web information systems. Lecture is held for students at Lübeck University of Applied Sciences (in german language). Lecture covers following aspects:

  • IaaS Cloud Computing to host (distributed and scalable) web applications
  • descriptional languages like HTML
  • presentational languages like CSS
  • Document Object Model (DOM) used for example in web browsers
  • client side programming with languages like JavaScript
  • server side programming with languages like PHP
  • security aspects like HTML/SQL injections or session hijacking

Course Material (german only)

  • Crash course: IaaS Cloud Computing
  • HTML Units: HTML and CSS
  • Database Units: Part 1, Part 2, Part 3
  • PHP Units: Part 1, Part 2, Part 3, Part 4, Part 5, Part 6
  • JavaScript Units: JavaScript
  • Exercises: Catalog of Excercises
